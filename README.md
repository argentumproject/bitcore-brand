##Bitcore

Bitcore is written with capital `B`. In code, `bitcore` is all lowercase.

- ✓ **Bitcore**
- ✓ **`bitcore`**
- × *BitCore*
- × *bitCorey*
- × *bit-core*
- × *bit core*
